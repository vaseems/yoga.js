Why Yoga.js?

Lightweight

No bloat, no fuss. Just clean, minimal extensions to core JS objects.

Future-proof

Polyfills & helpers that feel native. Use tomorrow’s JS today.

Readable

Make code more intuitive. Arrays with .first() and .last().

Code Example

import "yoga.js";

const arr = [1, 2, 3];
console.log(arr.first()); // 1
console.log(arr.last());  // 3

const obj = { a: 1, b: 2 };
console.log(obj.entries()); 
// [["a",1],["b",2]]
